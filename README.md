# GestBot

## Install

``` Bash
python -m pip install -r requirement.txt
```
make a **.env** file with
``` Env
SQL_HOSTNAME = "localhost"
SQL_PORT = "3306"
SQL_USERNAME = ""
SQL_USER_PASSWORD = ""
SQL_DB_NAME = "GestBot"
BOT_TOKEN = ""
```
