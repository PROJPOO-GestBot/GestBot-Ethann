# GestBot

## Install

``` Bash
python -m pip install -r requirement.txt
```
make a **.env** file with
``` Env
sqlHost = "localhost"
sqlPort = "3306"
sqlUser = ""
sqlPassword = ""
sqlDBName = ""
token = "Discord Token"
```
